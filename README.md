# Variables y Operadores
---
## Respuestas teóricas:  

- ¿Para qué se utilizan los operadores lógicos en programación?
	- Para controlar el flujo del programa y que se ejecuten las lineas que queramos dependiendo de las condiciones.
- ¿Por qué es importante declarar correctamente el tipo de dato de una variable?
	- Para evitar problemas con el formato (Principlamente decimales) y para aprovechar los recursos del sistema.

## Justificación Ténica:

- Clasificación de edad.
	- Para este use  _if-else_  porque me permitia comprender rangos de edades y no solo edades especificas como el  __switch__ .
- Día de la semana.
	- Usé el **switch**  ya que al ser un rango muy corto era muy facíl crear todos los casos requeridos para procesar los dias de la semana, ademas que mejora la visibilidad y eficiencia.
- Verificación de acceso.
	- En este caso usé ambas, el  **switch**  para los usuarios para rastrear las concidencias exactas y aunque podría haberlo utilizado también para las contraseñas se me hacia menos confuso para la legibilidad del codigo implementarla con el  __if-else__ .
